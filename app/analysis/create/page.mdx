# 분석 프로젝트 만들기

새로운 분석 프로젝트를 만드는 방법을 안내합니다.

분석 프로젝트를 생성하면 설문 응답 데이터를 기반으로 다양한 분석을 수행하고,  
그 결과를 보고서 형태로 정리하여 업무에 활용할 수 있습니다.

분석 목적과 데이터 준비 상태에 따라
아래 방법 중 하나를 선택해 분석을 시작하세요.

## 개요

데이터를 준비하여 분석 프로젝트를 만들고 분석을 통해 결과를 활용할 수 있습니다.

## 주요 기능

-   엑셀 파일로 프로젝트 만들기
-   설문 결과로 프로젝트 만들기

## 분석 생성

## 방법 1. 엑셀 파일로 프로젝트 만들기

외부에서 수집한 설문 데이터나
기존에 보유한 엑셀 파일을 활용해
와이즈온의 분석 기능을 사용하고 싶을 때 선택하는 방법입니다.

### 분석 생성 흐름

1. **생성하기** 메뉴 선택
2. **엑셀 파일로 프로젝트 만들기** 버튼 클릭
3. 프로젝트 필수 입력 사항 (**프로젝트명, 데이터 파일, 문항 속성 파일**)기입
4. **완료** 버튼 클릭

### 이런 경우에 적합합니다.

-   다른 설문 툴이나 오프라인 조사로 수집한 데이터가 있는 경우
-   기존에 진행했던 설문 결과를 분석만 새로 진행하고 싶은 경우
-   하나의 설문 데이터를 여러 분석 프로젝트로 관리하고 싶은 경우

<br />
## 방법 2. 설문 결과로 프로젝트 만들기

와이즈온에서 생성·배포한 설문의
응답 데이터를 바로 분석 프로젝트로 연결하는 방법입니다.

### 분석 생성 흐름

1. **생성하기** 메뉴 선택
2. **설문 결과로 프로젝트 만들기** 버튼 클릭
3. 설문 프로젝트 중에 분석에 사용 할 프로젝트 정하기
4. **분석하기** 버튼 클릭

### 이런 경우에 적합합니다.

-   와이즈온에서 설문 생성부터 배포까지 진행한 경우
-   설문 응답이 완료되자마자 바로 분석을 진행하고 싶은 경우
-   설문–응답–분석을 하나의 흐름으로 여러 분석 프로젝트를 관리하고 싶은 경우
