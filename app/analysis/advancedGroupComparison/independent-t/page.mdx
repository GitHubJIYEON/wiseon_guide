# **독립T검정분석(Independent Samples t-test)**

<br/>

## 1️⃣ 개요

**"두 집단이 서로 얼마나 다르게 응답했을까?"**

**"특정 집단이 다른 집단보다 더 만족하거나, 더 불만족한 항목은 무엇일까?"**

독립 t-검정은 **두 개의 서로 다른 집단 A와 B의 평균 차이를 비교**하는 통계 분석입니다.

예를 들어,

- 남성과 여성의 만족도 차이
- 재이용 고객 vs 신규 고객
- 20대 vs 40대의 서비스평가

처럼 **두 집단으로 나누어진 변수(=독립변수)**가 있을 때

각 집단의 평균 점수가 **통계적으로 유의미하게 다른지**를 검증합니다.

- **핵심 개념:** 두 집단 간 평균 차이 검증
- **주요 지표:** 평균(mean), t값(t-value), p값(p-value), 평균차(Mean Difference)
- **언제 사용하나요?**
    
    두 집단이 서로 다르게 느끼는 영역을 찾고, 개선 우선순위를 설정하고 싶을 때
    

<br/>

## 2️⃣ 왜 교차분석이 중요한가?

빈도나 평균만 보면

“차이가 조금 있다?” 정도만 알 수 있습니다.

하지만 t-검정을 하면 다음이 가능합니다:

### ✔ "차이가 우연인지, 실제로 존재하는 차이인지" 확인

p값을 기준으로 차이가 통계적으로 의미가 있는지 판단할 수 있습니다.

### ✔ "어느 집단이 더 높은지" 명확하게 확인

예:

- 재이용 고객의 만족도 평균 = 4.2
- 최초 이용 고객의 만족도 평균 = 3.6
    
    → 차이 + 방향성을 알 수 있음
    

### ✔ 정책·마케팅 타깃을 정확히 설정

집단별 차이를 근거로 “누가 문제를 더 느끼는지”를 판단할 수 있음

<br/>

## 3️⃣ 비즈니스 활용 예시

| 분야 | 활용 시나리오 | 얻게 되는 인사이트 |
| --- | --- | --- |
| **CX / 고객경험** | 재방문 고객 vs 신규 고객의 만족도 비교 | 충성 고객과 신규 고객의 차이 확인 |
| **공공기관 서비스 평가** | 지역별/성별 만족도 차이 검정 | 취약 집단의 불만 영역 파악 |
| **교육 평가** | 경력자 vs 비경력자의 강의 만족도 비교 | 교육 커리큘럼 난이도 조정 |
| **HR 조직진단** | 관리자 vs 직원의 조직문화 인식 비교 | 조직 내 인식 차이 원인 파악 |

## 4️⃣ 독립 t-검정 옵션 가이드

UI에 맞춰 정확히 작성했습니다.

### ① **독립변수(집단 변수) 선택 – 반드시 2집단만**

- 예:
    - 성별(남/여)
    - 재이용여부(Yes/No)
    - 전환의향(전환/미전환)
- 2개의 범주로 구성된 문항만 사용 가능
- 이 변수가 기준이 되어 **두 그룹을 나누는 역할**을 함

### ② **종속변수(비교할 문항) 선택 – 1개 이상 가능**

- 평균 점수로 비교 가능한 척도형 문항
- 예:
    - 만족도 모든 문항
    - 서비스평가 문항
    - 가격만족도
- 여러 문항을 한꺼번에 넣으면 그룹별 차이를 일괄 검정 가능

### ③ **표 형식 옵션**

- **행 기준**: 한 문항씩 행으로 나열
- **열 기준**: 그룹을 기준으로 문항 비교
    
    → 보고서나 대시보드 레이아웃에 맞게 선택
    

<br/>

## 5️⃣ 분석 단계 (Step-by-step)

교차분석 UI 흐름에 맞추어 단계별 작성했습니다.

1. **분석 메뉴 선택**
    - ‘집단비교분석 > 독립 t-검정’을 선택
2. **독립변수 선택**
    - 두 그룹으로 나뉘는 문항 선택 (예: 재이용여부)
3. **종속변수 선택**
    - 비교하고 싶은 점수형 문항 선택
    - 여러 개 선택 가능
4. **표 형식 선택**
    - 행기준 또는 열기준으로 결과 정렬
5. **분석하기 클릭**
6. **t값, p값, 평균차를 확인하며 결과 해석**

## 6️⃣ 결과 해석 (Interpretation)

독립 t-검정에서는 **3가지만** 보면 충분합니다.

### ✔ 1) p값(p-value): 차이가 있는가?

가장 중요한 지표.

| p값 | 해석 |
| --- | --- |
| **p < 0.05** | 두 집단의 차이가 통계적으로 ‘유의미함’ → **실제 차이가 존재** |
| p ≥ 0.05 | 차이가 통계적으로 확인되지 않음 → 우연적 차이일 가능성 높음 |

---

### ✔ 2) 두 집단의 평균 차이(Mean Difference): 어느 집단이 더 높은가?

- 평균이 상대적으로 높은 집단 → 더 긍정적인 평가를 하는 경향
- 평균이 상대적으로 낮은 집단 → 상대적으로 불만족하거나 경험이 좋지 않은 경향

예:

- 재이용 고객 만족도 = 4.3
- 신규 고객 만족도 = 3.8
    
    → “재이용 고객이 명확하게 더 만족하고 있음”
    

📌 주의
평균 차이가 크더라도 p값이 0.05 미만이 아닌 경우,
그 차이가 통계적으로 의미 있다고 말할 수 없습니다.
(즉, 우연일 가능성이 있다는 뜻)

---

### ✔ 3) 표준편차(SD): 그룹 내 의견 일치 정도

- SD가 낮으면 = 의견이 비슷함
- SD가 높으면 = 그룹 내부에서도 의견이 갈림

## 7️⃣ 자주 묻는 질문 (FAQ)

**Q. 독립 t-검정은 어떤 문항에 사용할 수 있나요?**

A. 평균이 의미 있는 **점수형(척도형) 문항**에서만 사용할 수 있습니다.

**Q. 집단이 3개 이상이면 사용할 수 있나요?**

A. 안 됩니다. 집단이 3개 이상이면 ANOVA(분산분석)를 사용해야 합니다.

**Q. 종속변수를 여러 개 넣어도 되나요?**

A. 가능합니다. 여러 만족도 문항을 한 번에 비교할 수 있어 실무에서 매우 유용합니다.

**Q. 표본 수가 두 집단에서 서로 크게 달라도 괜찮나요?**

A. 어느 정도 괜찮지만, 극단적으로 차이가 크면 p값 해석에 주의해야 합니다.

## 8️⃣ 보고서 작성 활용 문장 예시

- “재이용 고객은 신규 고객 대비 전반적 만족도가 유의미하게 높았다(p < 0.05).”
- “가격만족도 항목은 두 집단 간 차이가 나타나지 않았다(p ≥ 0.05).”
- “신규 고객의 직원친절 불만족 비율이 높아, 초반 서비스 경험 개선이 필요하다.”
- “20대와 40대 간 만족도 차이는 통계적으로 확인되지 않았다.”

🎯 최종 요약

우리 서비스의 t-검정 해석 로직은 다음 3단계입니다
1. 등분산 검정 확인
2. p값 확인 (p < 0.05이면 차이 있음)
3. 두 집단 평균 비교 (어느 집단이 더 높은지 파악)
Mean Difference가 없기 때문에
“평균 두 개를 직접 비교하는 방식”이 정답입니다.
