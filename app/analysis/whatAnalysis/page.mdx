## 개요

와이즈온 분석은 수집된 설문 데이터를 다양한 통계 기법으로 분석하여 인사이트를 도출하는 기능입니다.

## 주요 특징

-   기본 분석 (프로필, 리포팅, 교차분석)
-   고급 분석 (단변량, 집단비교, 척도화, 회귀분석)
-   분석 결과 보고서 생성
-   데이터 다운로드

## 분석 유형

### 기본분석

-   프로필 설정
-   기본 리포팅
-   교차분석

### 고급분석

-   단변량분석
-   집단비교분석
-   척도화분석
-   회귀분석
