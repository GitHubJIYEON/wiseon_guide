## 분석 프로세스 흐름

### 1. 데이터 준비

**방법 1. 엑셀 파일로 프로젝트 만들기**

**방법 2. 설문 결과로 프로젝트 만들기**

### 2-1. 기본 분석

분석 목적에 맞는 분석 방법 선택해 주세요.

-   프로필설정
-   기본리포팅
-   교차분석

### 2-2. 고급 분석

분석 목적에 맞는 분석 방법 선택해 주세요.

-   **단변량분석**
    -   빈도분석
    -   기술통계분석
    -   교차분석
-   **집단비교분석**
    -   독립T검정분석
    -   대응T검정분석
    -   일원배치분산분석
    -   이원배치분산분석
    -   공분산분석
-   **척도화분석**
    -   요인분석
    -   신뢰도분석
-   **회귀분석**
    -   상관관계분석
    -   선형회귀분석

### 2-3. 분석 방법 선택 가이드

-   **기술통계가 필요하면**: 기본분석 > 기본 리포팅
-   **집단 간 비교가 필요하면**: 고급분석 > 집단비교분석
-   **변수 간 관계를 파악하려면**: 고급분석 > 회귀분석
-   **척도를 검증하려면**: 고급분석 > 척도화분석

### 3. 분석 결과

-   분석 결과 확인
-   통계적 유의성 검토
-   인사이트 도출
