# 분석 전체 흐름 한눈에 보기

와이즈온 분석 프로세스의 전체 흐름을 단계별로 안내합니다.

## 분석 프로세스 흐름

### 1. 데이터 준비
- 설문 응답 데이터 확인
- 프로필 설정
- 데이터 품질 검토

### 2. 기본 분석
- 기본 리포팅 생성
- 교차분석 수행
- 기본 통계 확인

### 3. 고급 분석
- 분석 목적에 맞는 분석 방법 선택
- 단변량분석
- 집단비교분석
- 척도화분석
- 회귀분석

### 4. 결과 해석
- 분석 결과 확인
- 통계적 유의성 검토
- 인사이트 도출

### 5. 보고서 작성
- 분석 결과 보고서 생성
- 데이터 다운로드
- 결과 공유

## 분석 방법 선택 가이드

- **기술통계가 필요하면**: 기본분석 > 기본 리포팅
- **집단 간 비교가 필요하면**: 고급분석 > 집단비교분석
- **변수 간 관계를 파악하려면**: 고급분석 > 회귀분석
- **척도를 검증하려면**: 고급분석 > 척도화분석
