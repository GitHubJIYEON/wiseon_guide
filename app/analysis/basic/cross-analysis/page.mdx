## 개요

**"집단에 따라 응답 패턴이 달라지는가?"**

**"두 문항을 조합했을 때, 응답 분포는 어떤 차이를 보이는가?"**

교차분석은 두 개의 문항(변수)을 **행(Row)과 열(Column)로 교차 배치**해

응답 분포가 어떻게 달라지는지 비교하는 분석입니다.

쉽게 말해:

> 두 변수 간 관계를 '표' 형태로 시각적으로 확인해,
> 어떤 집단에서 어떤 응답이 두드러지는지 즉시 파악하는 분석 도구입니다.

-   **핵심 개념:** 분할표(cross table)를 활용한 응답 분포 비교
-   **주요 출력:** 빈도수, 백분율, 소수점 설정, 행·열 기준 시각표
-   **언제 사용하나요?:** “집단별 응답 경향 파악”, “문항 간 관계 확인”, “특정 그룹의 특이 패턴 탐색”이 필요할 때

---

## 1. 왜 교차분석이 중요한가?

교차분석은 단순한 빈도표로는 알 수 없는 **문항 간 상관 관계**를 드러내는 분석입니다.

특히 다음과 같은 질문에 직접적으로 답해줍니다:

-   담당 공무원의 **친절도에 따라** 민원인의 만족도가 달라지는가?
-   행정 서비스 평가 문항 간에는 어떤 **패턴의 일관성**이 있는가?
-   특정 문항에서 **불만족이 집중되는 특정 집단**이 있는가?
-   민원 처리 과정 만족도는 공무원의 **전문성과 연계**되어 나타나는가?

즉, 교차분석은

> “문항 간의 관계를 가장 명확하고 실무 친화적인 방식으로 보여주는 분석 도구”

이며

공공서비스 평가, CX 분석, 조직진단, UX 리서치 등 거의 모든 설문 분석에서 기본적으로 활용됩니다.

---

## 2. 비즈니스 활용 예시

| 분야                | 활용 시나리오                | 얻을 수 있는 인사이트                  |
| ------------------- | ---------------------------- | -------------------------------------- |
| **공공서비스 평가** | 민원 응대 경험 × 친절도      | 불만이 어느 집단에 집중되는지 확인     |
| **고객경험(CX)**    | 서비스 유형 × 만족도         | 특정 서비스 유형의 강·약점 파악        |
| **교육 품질 평가**  | 강사평가 × 과정전반 만족도   | 교육 효과에 영향을 주는 핵심 요인 탐색 |
| **HR/조직진단**     | 직급 × 조직문화 인식         | 조직 내 격차·오해 발생 지점 확인       |
| **제품/UX 조사**    | 기능 사용 여부 × 편의성 평가 | 기능 사용률이 낮은 원인 파악           |

## 3. 교차분석 옵션 설명 (UI 기준)

네 시스템의 교차분석은 매우 직관적이며 다음 세 구성이 핵심이다.

---

### ① 값 표시 옵션(Value Display)

-   **빈도수 + 백분율** (가장 일반적인 방식)
-   **백분율만**
-   **빈도수만**
-   **소수점 자릿수 설정**
    → % 비율 표현의 정밀도 조절 (예: 2자리 → 12.34%)

---

### ② 열 변수(Column) 선택

-   ‘기준이 되는 질문’을 선택
-   예: **"담당 공무원이 업무를 친절하게 처리했습니까?"**
-   각 열이 선택지(매우 불만족~매우 만족)를 나타냄

---

### ③ 행 변수(Row) 선택

-   비교하고 싶은 다른 설문 문항들
-   예: 응답자가 본 **표현의 적절성**, **명확성**, **절차 만족도** 등
-   여러 문항을 선택하면 아래로 쌓여 하나의 대형 교차표 생성

---

## 4. 분석 단계 (Step-by-step)

1. **분석 탭 선택**
    - ‘교차분석’ 메뉴 클릭
2. **값 표시 방식 선택**
    - 빈도수·백분율 옵션 및 소수점 설정
3. **열 변수 선택**
    - 기준 문항(예: 친절도, 공정성 등)
4. **행 변수 선택**
    - 비교할 여러 문항 선택
5. **분석하기 클릭**
    - 교차표(이미지 표 형태) 자동 생성

---

## 5. 결과 해석 (Interpretation)

교차분석은 아래 3가지를 중심으로 해석하면 매우 간단해진다.

---

### ✔ 1) 비율 분포 확인: 어떤 집단에서 응답이 몰리는가?

예:

-   “친절도 '매우 만족' 집단에서 전반 만족도가 45%로 가장 높음”
-   “불만족 응답은 특정 행 변수에서 10% 이상 집중됨”

이 단계에서 **이상 패턴(특정 집단에서 급증하는 응답)**을 찾기 좋다.

---

### ✔ 2) 문항 간 관계 탐색: 패턴이 일관적인가?

예:

-   친절도가 낮을수록 처리과정 ‘명확성’ 점수도 낮아진다
-   행정 절차 불만족이 공정성 낮음과 함께 나타난다

**문항 간 논리적 연결성**을 확인하는 단계다.

---

### ✔ 3) 정책/개선 포인트 도출

-   불만족이 특정 집단에서 집중 → 해당 집단을 중심으로 개선 필요
-   '매우 만족' 비율이 특정 행 변수에서 높음 → 강점 요소

이렇게 교차표는 **문항 단위로 문제 영역을 pinpoint**하는 데 매우 유용하다.

---

## 6. 자주 묻는 질문 (FAQ)

### **Q. 어떤 문항을 열 변수로 넣어야 하나요?**

A. 기준이 되는 핵심 문항(예: 전체 만족도, 친절도 등)을 열에 배치하는 것이 해석이 가장 쉽습니다.

### **Q. 행 변수는 여러 개 선택해도 되나요?**

A. 가능합니다. 여러 문항을 한 번에 비교할 수 있습니다.

### **Q. 백분율은 행 기준인가요? 열 기준인가요?**

A. 네 UI에서는 ‘전체 응답 대비 비율’을 주로 표시합니다.

행 기준/열 기준 모두 직관적 비교가 가능하도록 설계되어 있음.

### **Q. 어떤 상황에서 교차분석을 사용해야 할까?**

A. 모든 ‘문항 간 관계’를 보고 싶을 때 가장 먼저 고려하는 분석입니다.
