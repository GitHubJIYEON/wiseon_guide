**“응답자들은 어떤 사람들일까요?”**

프로필 분석은 설문에 참여한 **응답자의 기본 특성 분포를 파악**하는 분석입니다.

성별, 연령대, 직무, 지역, 이용 경험 여부 등과 같은 **배경 정보(Profile 변수)**를 통해

**누가 응답했는지**, 그리고 **응답 집단의 구성은 어떤지**를 한눈에 확인할 수 있습니다.

-   **핵심 개념:** 응답자 구성 분포 (Respondent Distribution)
-   **주요 지표:** 빈도수(Frequency), 백분율(Percentage)
-   **언제 사용하나요?:**
    -   설문 결과 해석 전에 **표본의 특성**을 먼저 이해하고 싶을 때
    -   특정 집단(연령, 성별 등)에 **응답이 치우치지 않았는지** 확인할 때
    -   보고서 서두에 **응답자 특성 요약**이 필요할 때

---

## 2. 비즈니스 활용 예시

프로필 분석은 모든 설문 분석의 **출발점**입니다.

“이 결과를 **누가** 말했는가?”를 설명하지 못하면, 해석의 설득력이 떨어집니다.

| **분야**           | **활용 시나리오**                    | **기대 효과**                                   |
| ------------------ | ------------------------------------ | ----------------------------------------------- |
| **마케팅**         | 고객 설문 응답자 성별·연령 분포 확인 | 결과가 **목표 고객층을 제대로 반영했는지 검증** |
| **HR (조직진단)**  | 부서·직급별 응답 비율 분석           | 특정 조직에 **응답 편중 여부 확인**             |
| **공공·정책 조사** | 지역·연령대 구성 확인                | 정책 수요를 **어떤 집단이 대표하는지 파악**     |
| **보고서 작성**    | 응답자 특성 요약 페이지              | 분석 결과의 **신뢰성과 맥락 강화**              |

## 3. 필요한 옵션 가이드

프로필 분석은 분석 모델이 단순하지만, **표현 방식 설정이 매우 중요**합니다.

### ① 갑 표시 설정

각 응답 항목의 값을 **어떤 숫자 형태로 보여줄지** 선택합니다.

-   **빈도수 + 백분율** **[기본값 / 추천]**
    -   예: `120명 (48.6%)`
    -   가장 직관적인 표현 방식
-   **백분율**
    -   예: `48.6%`
    -   비율 비교 중심의 보고서에 적합
-   **빈도수**
    -   예: `120명`
    -   실제 응답 규모를 강조하고 싶을 때 사용

---

### ② 소수점 설정

백분율 표시 시 **소수점 자릿수**를 설정합니다.

-   선택 범위: **0 ~ 3 자리**
-   예시
    -   소수점 0 → `49%`
    -   소수점 1 → `48.6%`
    -   소수점 2 → `48.62%`

---

### ③ 표기 옵션

분석 결과를 **어떤 형태의 보고서로 생성할지** 선택합니다.

-   **막대그래프**
-   **해석**
-   **빈도표**

> ✔️ 선택한 옵션에 따라 해당 형식의 결과 보고서가 자동 생성됩니다.
>
> (복수 선택 가능)

---

## 4. 분석 단계 (Step-by-step)

_(이곳에는 실제 프로젝트의 UI 스크린샷을 배치하고 번호를 매겨 설명합니다.)_

1. **프로필 변수 선택**
    - 성별, 연령대, 직급, 지역 등 분석할 프로필 변수를 선택합니다.
    - _Tip: 분석 목적에 따라 불필요한 변수는 제외하는 것이 좋습니다._
2. **표시 옵션 설정**
    - 갑 표시, 소수점, 표기 옵션을 설정합니다.
3. **분석 실행**
    - 실행 버튼을 클릭하면 선택한 형식의 프로필 분석 보고서가 생성됩니다.

## 5. 결과 해석 (Interpretation)

프로필 분석 결과는 **“누가 응답했는가?”**를 설명하는 데 집중합니다.

### Check 1. 응답 분포 확인

-   특정 항목에 **응답이 과도하게 몰려 있지 않은지** 확인합니다.
-   예:
    -   20대 응답자 85%
        → 결과 해석 시 **연령 편향 가능성** 고려 필요

---

### Check 2. 보고서 표현 방식 이해

-   **막대그래프**
    -   항목 간 비율 차이를 한눈에 비교 가능
-   **빈도표**
    -   정확한 수치 확인 및 내부 검토용으로 적합
-   **해석**
    -   “응답자의 절반 이상이 ○○에 해당합니다.” 와 같은
        **자동 해석 문장 제공**

---

## 6. 자주 묻는 질문 (FAQ)

**Q. 프로필 분석은 꼭 해야 하나요?**

A. 네. 프로필 분석은 모든 설문 분석의 **기본 전제**입니다.

응답자 특성을 모르면 이후 분석 결과의 해석이 왜곡될 수 있습니다.

---

**Q. 프로필 변수도 교차분석이나 회귀분석에 사용할 수 있나요?**

A. 가능합니다.

프로필 분석은 **단독으로도 사용**되지만,

이후 교차분석·집단비교·회귀분석의 **독립변수 후보**로 활용됩니다.

---

**Q. 응답자가 너무 한쪽으로 몰려 있으면 어떻게 하나요?**

A. 결과 자체가 틀린 것은 아닙니다.

다만 보고서에

> “응답자가 특정 집단에 편중되어 있어 해석에 유의가 필요함”
>
> 과 같은 **해석 주석을 함께 제시**하는 것이 바람직합니다.

---

### 한 줄 요약

> 프로필 분석은 ‘이 설문 결과를 누가 말했는지’를 설명하는 분석입니다.
>
> 모든 통계 분석의 출발점이자, 보고서 신뢰도를 결정하는 핵심 단계입니다.
