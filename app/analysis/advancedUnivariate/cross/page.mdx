# 교차분석

<br/>

## 1️⃣ 개요

**"집단에 따라 응답 패턴이 다를까?"
"특정 문항의 응답이 다른 문항과 어떤 관계가 있을까?"**

교차분석은 두 개의 문항(변수)을 행(row)과 열(column)에 놓고,

응답이 어떻게 결합되어 나타나는지 비교하는 분석입니다.

쉽게 말해,

**A 그룹과 B 그룹이 서로 다르게 응답했는지**

또는

**특정 선택지가 특정 집단에서 더 많이 나타나는지**

확인하는 방법입니다.

- **핵심 개념:** 분할표(cross table)를 통한 패턴 비교
- **주요 지표:** 빈도, 비율(행비율/열비율), 카이제곱(Chi-square) 검정
- **언제 사용하나요?:** "누가 그렇게 답했는가?" "어떤 그룹이 특히 높은가?"를 알고 싶을 때

<br/>

## 2️⃣ 왜 교차분석이 중요한가?

교차분석은 설문 결과에서 **집단 간 차이를 발견하는 가장 기본적인 방식**입니다.

단순 빈도·평균만으로는 알 수 없는 ‘왜 이런 결과가 나왔는가?’에 대한 힌트를 제공합니다.

다음 질문들에 명확하게 답할 수 있습니다.

- 남성과 여성의 만족도는 다를까?
- 고객경험이 많은 사람일수록 추천 의향이 높은가?
- 불만족 응답은 특정 연령대에서 집중되었나?
- A지점과 B지점 고객의 평가가 어떻게 다른가?
- 서비스 재이용 의향이 낮은 집단은 누구인가?

즉, 교차분석은 **문제의 원인을 찾는 분석**으로 이어지는 매우 중요한 단계입니다.

<br/>

## 3️⃣ 비즈니스 활용 예시

교차분석은 대부분의 실무 분석에서 핵심적으로 활용됩니다.

| **분야** | **활용 시나리오** | **얻을 수 있는 인사이트** |
| --- | --- | --- |
| **공공서비스 평가** | 지역별·성별 만족도 차이 비교 | 특정 집단의 불만 요인 파악 |
| **고객경험(CX)** | 재방문 의향 × 고객유형 비교 | 충성도가 높은/낮은 고객 식별 |
| **제품 기획** | 기능 선호도 × 연령대 분석 | 타깃 세그먼트 정의 |
| **HR 조직진단** | 직급 × 조직문화 인식 | 갈등 또는 격차 영역 탐색 |
| **교육/훈련 평가** | 학습자 유형 × 강의만족도 | 특정 집단의 낮은 만족 원인 발견 |

## 4️⃣ 교차분석 옵션 가이드

UI에 맞춰 정확히 작성했습니다.

### ① **통계량 선택**

- **빈도(Count)**
    - 실제 응답 수를 그대로 보여줍니다.
    - 표의 기본 형태.
- **비율(%)**
    - 비율 중심 분석. 전체 비율을 기준으로 비교하기 좋음.
- **빈도 + 비율**
    - 가장 실무에서 많이 사용. 숫자와 비율을 동시에 보여줌.

### ② **백분율 기준 선택 (핵심 옵션)**

### ✔ 행비율(Row %)

- “행 기준으로 각 열이 얼마나 차지하는가?”
- 예: 성별 → 만족도 보면
    - 남성 60%가 ‘만족’, 여성 40%가 ‘만족’ → 성별 차이 분석에 최적

### ✔ 열비율(Column %)

- “열 기준으로 각 행이 어떤 비중을 차지하는가?”
- 예: 만족도 → 성별 보면
    - ‘만족한 사람 중 남성 55%, 여성 45%’
- 특정 응답의 구성비를 보고 싶을 때 활용

> Tip: '집단별 응답 비교'는 행비율을, ‘응답의 구성비 이해’는 열비율을 사용합니다.
> 

<br/>

## 5️⃣ 분석 단계 (Step-by-step)

교차분석 UI 흐름에 맞추어 단계별 작성했습니다.

1. **분석 메뉴 선택**
    - 상단에서 ‘단변량분석 > 교차분석’을 클릭합니다.
2. **행 변수 선택 (Row)**
    - 집단 구분 변수 선택
    - 예: ‘성별’, ‘방문경험’, ‘재이용 의향’
3. **열 변수 선택 (Column)**
    - 비교할 응답 변수 선택
    - 예: ‘직원친절만족’, ‘요금제도만족’, ‘영화관만족’
4. **통계량 옵션 설정**
    - 빈도, 비율, 빈도+비율 선택
    - 백분율 기준(행·열) 선택
5. **분석 실행**
    - ‘분석하기’ 버튼을 클릭하여 결과 확인
6. **결과 해석**
    - 셀 값(빈도·비율)을 기준으로 집단 간 차이를 파악
    - 필요하면 ‘행/열 기준 변경’하면서 패턴을 비교

## 6️⃣ 결과 해석 (Interpretation)

교차분석은 아래 3가지를 중심으로 보면 됩니다.

### ✔ 1) 집단 간 비율 차이

행비율 기준으로 각 셀을 보면

“어떤 그룹이 어떤 응답을 더 많이 했는가?”를 확인할 수 있습니다.

예)

- 남성의 70%는 ‘만족’, 여성의 55%는 ‘만족’ → 성별 차이 존재

### ✔ 2) 응답 분포의 편중 여부

특정 집단에서 특정 응답이 지나치게 많으면 문제의 원인을 찾는 실마리입니다.

예)

- 50대의 30%가 ‘불만족’ → 나이 기반 서비스 개선 필요

### ✔ 3) 세그먼트 발견

데이터 안의 특별한 패턴을 찾아낼 수 있습니다.

예)

- ‘재방문 의향 낮음’ + ‘직원친절 불만족’ → 불만족 요인의 주요 원인

## 7️⃣ 자주 묻는 질문 (FAQ)

**Q. 교차분석은 어떤 문항에서 사용할 수 있나요?**

A. 범주형(객관식), 단일선택 문항이라면 대부분 가능하며

점수형 문항도 구간화한 경우(예: 1~5점) 사용 가능합니다.

**Q. 행·열 변수는 어떻게 선택해야 하나요?**

A.

- **행(Row): 집단을 구분하는 변수**(성별/연령/지점 등)
- **열(Column): 결과를 비교하고 싶은 변수**(만족도/의향 등)
    
    이 가장 자연스럽습니다.
    

**Q. 비율과 행/열 기준 중 무엇을 사용해야 하나요?**

A.

- **집단별 차이 비교 → 행비율**
- **응답 구성비 파악 → 열비율**
- 실무에서는 빈도+행비율 조합이 가장 많이 사용됩니다.

**Q. 카이제곱검정(Chi-square test)도 제공되나요?**

A. 실무에서 많이 쓰이지만, UI 단순화를 위해 기본 표 중심으로 제공할 수 있습니다.

필요 시 자동 제공 기능 확장 가능합니다.

## 8️⃣ 보고서 작성 활용 문장 예시

- “재방문 의향은 최근 방문 여부에 따라 유의한 차이를 보였다. 최근 방문 고객의 72%가 ‘재방문 의향 있음’으로 응답하였다.”
- “요금 만족도는 연령대에 따라 뚜렷한 차이를 보였으며, 50대의 불만족 비율이 가장 높았다.”
- “직원 친절성 불만은 특정 지점(A지점)에 집중되어 나타났다.”
- “전환 의향은 서비스 경험 수준에 따라 달라졌으며, 경험이 많은 사용자일수록 이탈률이 낮았다.”
