## 개요

작성한 설문을 응답자에게 배포하는 방법을 안내합니다.

## 배포 방법

### ✔️ 익명 링크

**익명 링크란?**

익명 링크는 개인 정보 없이 설문을 배포하는 방식입니다.
하나의 URL로 여러 응답자가 참여할 수 있으며, 응답자는 완전히 익명으로 설문에 참여할 수 있습니다.

**익명 링크 사용 방법**

1. 상단 메뉴에서 [배포] 탭 선택
2. 왼쪽 메뉴에서 익명 링크 선택
3. 생성된 설문 URL을 복사
4. 문자, 메신저, 이메일, 홈페이지 등 원하는 채널로 링크 붙여넣기 배포

**익명 링크 특징**

-   **로그인 없이 참여 가능** : 응답자는 별도의 로그인 과정 없이 설문에 바로 참여할 수 있습니다.
-   **응답 데이터만 수집** : 응답 내용만 저장되며, 응답자의 신원 정보는 수집되지 않습니다.
-   **대규모 설문에 적합** : 불특정 다수를 대상으로 한 설문이나 공개 설문에 적합합니다.

**사용 시 참고 사항**

미리보기 링크에서 입력한 응답은 실제 데이터로 저장되지 않습니다.
익명 링크는 개인별 응답자 추적이 불가능합니다.
응답자별 관리가 필요한 경우에는 개인 링크 배포를 권장합니다.

---

### ✔️ 개인 링크

특정 응답자에게만 발급되는 개인별 고유 링크입니다.

**개인 링크란?**

개인 링크는 응답자별로 고유한 설문 URL을 생성하여,
누가 응답했는지 시스템에서 구분·관리할 수 있는 배포 방식입니다.
응답자의 신원 정보와 설문 결과를 정확하게 매칭할 수 있어,
대상자별 분석이 필요한 설문에 적합합니다.

-   응답자마다 고유한 설문 URL이 생성됩니다.
-   응답 결과는 대상자 정보와 함께 매칭되어 저장됩니다.
-   별도의 로그인 없이도 개인별 응답 식별이 가능합니다.

**개인 링크 사용 방법**

1. 상단 메뉴에서 **[배포]** 탭 선택
2. 왼쪽 메뉴에서 **개인 링크** 선택
3. **[링크 생성]** 버튼 클릭
4. 개인 링크를 공유 할 정보 입력 또는 엑셀 파일로 데이터 등록
    - 개인 링크 생성 시, 제목
    - 개인 정보 입력: 휴대폰번호, 이메일, 이름 등
5. **저장** 버튼 클릭 → 개인 링크 생성 완료
