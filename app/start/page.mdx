와이즈온(WiseON)은

**설문 생성 → 배포 → 응답 수집 → 분석 → 보고서**까지
설문 업무 전 과정을 하나의 흐름으로 관리할 수 있는 플랫폼입니다.

모든 작업은 하나의 **설문 프로젝트**를 중심으로 연결되며,
설문 결과 데이터를 기반으로 다양한 분석을 수행하여
의미 있는 결과물을 도출하고 실제 업무에 적용할 수 있습니다.

## 와이즈온 사용 흐름 한눈에 보기

와이즈온의 기본 사용 흐름은 아래와 같습니다.

### 설문

- 설문 프로젝트 생성
    - 문항 설계
    - 설문 미리보기 및 테스트
- 설문 배포
- 응답 수집 및 현황 확인
- 설문 프로젝트 목록 관리

### 분석

- 기본 분석
    - 프로필설정
    - 기본리포팅
    - 교차분석
- 고급 분석
    - 단변량분석 : 빈도분석, 기술통계분석, 교차분석
    - 집단비교분석 : 독립T검정분석, 대응T검정분석, 일원배치분산분석, 이원배치분산분석, 공분산분석
    - 척도화분석 : 요인분석, 신뢰도분석
    - 회귀분석 : 상관관계분석, 선형회귀분석,
- 결과 다운로드 및 보고서 활용

이 가이드는 위 흐름을 기준으로 구성되어 있으며,
순서대로 따라가면 설문과 분석을 자연스럽게 완료할 수 있습니다.

## 처음이라면 이렇게 시작하세요

와이즈온을 처음 사용하신다면 아래 순서를 추천합니다.

**1️⃣ 설문 프로젝트 생성하기**
신규 설문을 만들거나, 템플릿을 활용해 빠르게 시작할 수 있습니다.

**2️⃣ 문항 추가 및 설계**
객관식, 입력형, 척도형 등 다양한 문항 유형을 활용해 설문을 구성합니다.

**3️⃣ 미리보기로 설문 확인**
실제 응답자 화면을 미리 확인하고 오류를 점검합니다.

**4️⃣ 설문 배포**
익명 링크 또는 개인 링크를 통해 설문을 배포합니다.
